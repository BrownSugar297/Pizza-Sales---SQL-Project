# 🍕 Pizza Sales Analysis (SQL Project)

## 📌 Project Overview
This project focuses on analyzing a pizza sales dataset using SQL to extract meaningful business insights. It demonstrates data querying, aggregation, and analysis skills that are essential for data analysts and AI/ML engineers working with structured data.

The project is divided into **Basic, Intermediate, and Advanced SQL queries**, showcasing progressive learning and real-world analytics use cases.

---

## 🎯 Objectives

### 🔹 Basic Analysis
- Retrieve the total number of orders placed
- Calculate total revenue generated
- Identify the highest-priced pizza
- Determine the most common pizza size ordered
- Find the top 5 most ordered pizza types

### 🔹 Intermediate Analysis
- Calculate total quantity of each pizza category
- Analyze order distribution by hour of the day
- Find category-wise pizza distribution
- Compute average pizzas ordered per day
- Identify top 3 pizzas based on revenue

### 🔹 Advanced Analysis
- Calculate percentage contribution of each pizza to total revenue
- Find top 3 pizzas by revenue for each category
